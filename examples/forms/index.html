Native.js Form Components

Pre-built form components for common patterns. Just define the URL and fields!

Search / Filter Form (<n-fetch-form>)

Use for search fields, filters, or any form that fetches data based on input.

Live Example

Enter a search term and click Search 
<n-fetch-form n-action="/api/search" n-state-key="searchResults" n-target="search-results">
    <input type="search" name="q" placeholder="Search...">
    <button type="submit">Search</button>
</n-fetch-form>

<div id="search-results"></div>

Attributes

  • n-action - URL to fetch from (required)
  • n-state-key - Key to store results in state (default: "data")
  • n-target - ID of element to receive n-data-received event

Contact Form (<n-submit-form>)

Use for contact forms, registration, or any form that submits data.

Live Example

<n-submit-form 
    n-action="/api/contact" 
    n-method="POST"
    n-success-message="Thank you! Your message has been sent.">
    
    <input type="text" name="name" placeholder="Name" required>
    <input type="email" name="email" placeholder="Email" required>
    <textarea name="message" placeholder="Message"></textarea>
    <button type="submit">Send</button>
</n-submit-form>

Attributes

  • n-action - URL to submit to (required)
  • n-method - HTTP method: POST, PUT, PATCH, DELETE (default: POST)
  • n-reset - Reset form after success (default: true)
  • n-success-message - Message shown on success
  • n-error-message - Message shown on error

Working with Events

Both form components dispatch events you can listen to:

// Fetch Form Events
document.querySelector('n-fetch-form').addEventListener('n-fetch-success', (e) => {
    console.log('Data received:', e.detail.data);
});

document.querySelector('n-fetch-form').addEventListener('n-fetch-error', (e) => {
    console.log('Fetch failed:', e.detail.error);
});

// Submit Form Events  
document.querySelector('n-submit-form').addEventListener('n-submit-success', (e) => {
    console.log('Submitted:', e.detail.data);
});

document.querySelector('n-submit-form').addEventListener('n-submit-error', (e) => {
    console.log('Submit failed:', e.detail.error);
});
